ShopEZ – E-commerce Frontend 

ShopEZ is a responsive e-commerce web application frontend developed using React, Vite, and Tailwind CSS. It provides a simple online shopping interface where users can browse products, view details, add items to cart, and place orders through a clean and mobile-friendly UI.

This project was created as part of a college team assignment (Skill Wallet – E-commerce Application) to understand MERN stack architecture and frontend development.

Features Product listing and product details Add to cart and quantity update Cart total calculation Basic checkout flow Responsive layout (mobile & desktop) Navigation between pages Search and product browsing Tech Stack React.js Vite Tailwind CSS JavaScript React Router DOM Axios Project Structure

Installation 

Install dependencies:

npm install

Start development server:

npm run dev

Frontend runs on:

http://localhost:5173

Backend Connection 

The frontend connects to backend API using:

VITE_BACKEND_URL=http://localhost:4000

Academic Context 

This frontend is part of the ShopEZ MERN e-commerce project developed as a team assignment to demonstrate full-stack web development and React-based UI design.
